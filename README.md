# WeatherStation
Weather station using Arduino
The goal of this project is to allow a termometer accuracy. It is composed of a screen, a wemos D1 and a web interface to check for the accuracy and set the BPM.<BR>

## Needed hardware <BR>
- Weemos D1 Mini<BR>
